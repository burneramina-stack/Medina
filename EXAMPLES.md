## Пример 1: Добавление исторического события

**Задача**: Записать информацию о начале Первой чеченской войны

**Шаги**:

1. Копирую шаблон:
```bash
cp research/history/_template.md research/history/1994-начало-первой-войны.md
```

2. Заполняю файл информацией (см. example в `research/history/`)

3. Добавляю в хронологию:
```markdown
## 1994-1996 (Первая чеченская война)

### 11 декабря 1994 - Начало войны
- Ввод федеральных войск в Чечню
- [[research/history/1994-начало-первой-войны.md]]
```

4. Добавляю теги: `#история #1994 #первая_война`

## Пример 3: Создание главы с исследованиями

**Задача**: Написать главу с описанием Грозного в 1995 году

**Шаги**:

1. Сначала собираю исследования:
   - Создаю `research/geography/grozny-1995.md`
   - Создаю `research/history/1995-grozny-situation.md`
   - Добавляю релевантные слова в словарь

2. Создаю главу:
```bash
cp manuscript/chapters/_template.md manuscript/chapters/chapter-03.md
```

3. В файле главы ссылаюсь на исследования:
```markdown
## Исследовательские материалы
- [[research/geography/grozny-1995.md]] - описание разрушенного города
- [[research/history/1995-grozny-situation.md]] - исторический контекст
- [[research/language/vocabulary.md]] - чеченские названия улиц
```

## Пример 4: Организация свидетельства

**Задача**: Записать историю из интервью

**Шаги**:

1. Копирую шаблон:
```bash
cp research/testimonies/_template.md research/testimonies/2024-интервью-001.md
```

2. Заполняю с учетом этики:
```markdown
## Источник
- **Тип**: интервью
- **Автор/рассказчик**: [АНОНИМНО - И.М.]
- **Дата**: 15 марта 2024
- **Место**: [не указано]

## Этические соображения
- Анонимность: ✓ Обязательна
- Чувствительность: Высокая - упоминания насилия
- Согласие на использование: Получено устно
```

## Пример 5: Связывание материалов

**Сценарий**: У меня есть сцена в главе, которая требует нескольких исследований

В файле главы:
```markdown
## Сцена: Встреча на рынке в 1996

Исследования:
- [[research/culture/traditions.md#рынок]] - поведение на рынке
- [[research/language/vocabulary.md#торговля]] - фразы для торговли
- [[research/geography/locations.md#грозный]] - описание места
- [[research/history/1996-междувоенный-период.md]] - контекст эпохи
```

При написании могу быстро открыть все нужные файлы.

## Пример 7: Рабочий процесс для главы

**Полный цикл написания главы**:

1. **Идея** → Записываю в `notes/ideas.md`:
```markdown
### Глава 5 - Возвращение домой
- Главный герой возвращается в разрушенную деревню
- Нужно: описание деревни, традиции возвращения, эмоции
```

2. **Исследование** → Создаю файлы:
   - `research/geography/село-алхан-юрт.md`
   - `research/culture/traditions.md` (раздел о возвращении)

3. **Хронология** → Проверяю `timeline.md` для исторического контекста

4. **Написание** → Создаю `chapter-05.md` и пишу, ссылаясь на исследования

5. **Редактура** → Проверяю факты по источникам в исследованиях

## Советы по эффективности

### Быстрое создание файла
Вместо копирования, можно сразу создать файл с основной структурой:
```bash
# Создаю файл события
echo "# Битва за Грозный 1994-1995" > research/history/битва-грозный.md
# Потом добавляю содержимое из шаблона
```

### Групповая работа с тегами
Добавляю несколько тегов к одному файлу:
```markdown
## Теги
#история #первая_война #грозный #1994 #1995 #городской_бой
```

### Перекрестные ссылки
Создаю сеть знаний:
```markdown
## Связанные события
- [[1991-распад-ссср.md]] - предпосылки
- [[1996-хасавюртовские-соглашения.md]] - последствия
- [[biography-дудаев.md]] - ключевая фигура
```

## Частые вопросы

**Q: Где хранить черновики текста?**
A: В `manuscript/chapters/` - каждая глава отдельным файлом

**Q: Как организовать большое количество свидетельств?**
A: Нумеруйте их: `2024-интервью-001.md`, `2024-интервью-002.md` и т.д.

**Q: Нужно ли заполнять все поля в шаблоне?**
A: Нет, заполняйте только релевантные поля

**Q: Можно ли создавать свои категории в research/?**
A: Да! Создавайте подпапки по необходимости
