# Medina

> Художественная книга о Чечне и травме

---

## О проекте

Это исследовательский и писательский проект, посвященный созданию художественной книги о Чечне, травме и памяти.

Проект объединяет:
- Глубокое историческое и культурное исследование
- Художественное повествование
- Систематическую организацию материалов

---

## Структура

Проект организован для эффективной работы с исследованиями и написанием:

- **research/** - Исследовательские материалы по категориям
- **manuscript/** - Рукопись книги
- **notes/** - Идеи, заметки, планы
- **timeline.md** - Хронология исторических событий

Подробная навигация: [INDEX.md](INDEX.md)

---

## Как начать

1. **Изучите структуру**: откройте [INDEX.md](INDEX.md) для полной карты проекта
2. **Заполняйте исследования**: используйте шаблоны в папках `research/`
3. **Ведите хронологию**: добавляйте важные даты в [timeline.md](timeline.md)
4. **Собирайте идеи**: записывайте мысли в [notes/ideas.md](notes/ideas.md)
5. **Пишите главы**: создавайте файлы в `manuscript/chapters/`

---

## Ключевые принципы

1. **Точность**: всегда указывайте источники для фактов
2. **Этика**: обращайтесь с чувствительными материалами уважительно
3. **Связность**: используйте ссылки между файлами для создания сети знаний
4. **Систематичность**: используйте шаблоны и теги

---

## Примеры использования

### Добавить историческое событие
```bash
# Скопируйте шаблон
cp research/history/_template.md research/history/1994-штурм-грозного.md

# Заполните информацию
# Добавьте событие в timeline.md
```

### Создать новую главу
```bash
# Скопируйте шаблон
cp manuscript/chapters/_template.md manuscript/chapters/chapter-01.md

# Напишите главу, ссылаясь на исследования
```

### Добавить чеченское слово
```bash
# Откройте словарь
# research/language/vocabulary.md
# Добавьте слово в соответствующую таблицу
```
