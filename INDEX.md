# Medina - Навигация по проекту

> Художественная книга о Чечне и травме

---

## Быстрый старт

### Начинаю новое исследование
1. Выберите категорию в `research/`
2. Скопируйте `_template.md` и переименуйте
3. Заполните шаблон
4. Добавьте в хронологию `timeline.md` если нужно

### Пишу новую главу
1. Скопируйте `manuscript/chapters/_template.md`
2. Назовите файл как `chapter-01.md`, `chapter-02.md` и т.д.
3. Добавьте ссылки на нужные исследования

---

## Система тегов

Используйте теги в конце файлов для связи материалов:

### Категории
- `#история` `#политика` `#культура` `#язык` `#география`

### Периоды
- `#до1991` `#1991-1994` `#первая_война` `#1996-1999` `#вторая_война` `#2000е` `#2010е`

### Темы
- `#травма` `#память` `#идентичность` `#семья` `#сопротивление`

### Персонажи
- `#персонаж_имя` (добавляйте по мере создания)

## Советы по работе

1. **Связывайте материалы**: используйте `[[имя_файла.md]]` для ссылок между документами
2. **Проверяйте факты**: всегда указывайте источники
3. **Этика**: помните о чувствительности темы, особенно при работе со свидетельствами
4. **Регулярные бэкапы**: делайте git commit после важных добавлений

---

## Рабочий процесс

1. **Исследование** → Создаю файл в `research/` с фактами и источниками
2. **Хронология** → Добавляю в `timeline.md` для контекста
3. **Заметки** → Записываю идеи в `notes/ideas.md`
4. **Написание** → Создаю главу в `manuscript/chapters/`, ссылаюсь на исследования
5. **Редактура** → Проверяю факты, улучшаю текст
